# HashMap em Java

Exemplo simples de utilização da classe `HashMap` para armazenar IDs e nomes de clientes.

## Funcionalidades

- Criar um `HashMap`
- Adicionar clientes
- Buscar um cliente pelo ID
- Exibir o resultado no console

## Como executar

1. Compile o arquivo:
   ```bash
   javac Main.java
   ```

2. Execute:
   ```bash
   java Main
   ```

## Saída esperada

```text
O nome do cliente com ID 2 é: Marcos
```
